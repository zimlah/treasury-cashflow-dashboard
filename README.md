# Treasury Cash Flow Automation Dashboard

## Overview
This project demonstrates an automated treasury reporting system built in Microsoft Excel.

## Features
- Daily cash flow tracking
- Liquidity position monitoring
- Vendor payment reconciliation
- KPI performance dashboard
- Structured financial reporting templates

## Tools Used
- Microsoft Excel (Advanced Formulas)
- Financial Modeling
- Data Structuring & Automation

## Objective
To reduce manual treasury reporting time and improve financial visibility for decision-making.
